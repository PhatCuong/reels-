# Reels Automation

Public verification site for Reels Automation.

## TikTok developer verification URLs

After enabling GitHub Pages for this repository, use:

- App / Website URL: `https://phatcuong.github.io/reels-/`
- Terms of Service URL: `https://phatcuong.github.io/reels-/terms.html`
- Privacy Policy URL: `https://phatcuong.github.io/reels-/privacy.html`

## GitHub Pages setup

1. Open this repository on GitHub.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/** root
4. Save.
5. Wait 1-3 minutes, then open `https://phatcuong.github.io/reels-/`.

## TikTok platform choice

For this automation, choose **Web** unless TikTok specifically requires another platform for your product.
